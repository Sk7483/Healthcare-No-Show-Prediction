# 🏥 Healthcare No-Show Prediction

## 📌 Project Overview

This project predicts whether a patient will miss a scheduled medical appointment using Machine Learning. Early prediction of no-shows helps healthcare providers improve scheduling efficiency, reduce resource wastage, and enhance patient care.

## 🎯 Objective

Develop a predictive model that identifies patients who are likely to miss their appointments based on demographic and appointment-related information.

## 📊 Dataset

- Dataset: KaggleV2-May-2016
- Records: 110,000+ patient appointments
- Features include:
  - Gender
  - Age
  - Neighbourhood
  - Scholarship
  - Hypertension
  - Diabetes
  - Alcoholism
  - Handicap
  - SMS Received
  - Appointment Details

## 🤖 Machine Learning Model

**Random Forest Classifier**

The Random Forest algorithm was used to classify whether a patient would attend or miss an appointment.

## 📈 Results

- Model: Random Forest Classifier
- Accuracy: **77%**
- Successfully predicted patient appointment attendance behavior.
